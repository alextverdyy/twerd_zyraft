# Leader sequences

Press the `D+F` combo to start Leader, then enter a sequence. `S+D+F` starts Leader with sticky Shift. Shift keys are ignored while a sequence is being recognized.

## German characters

| Sequence | Output |
|---|---|
| `A` | ä |
| `O` | ö |
| `U` | ü |
| `S` | ß |

## Greek characters

All Greek sequences start with `E`.

| `E A` | α | `E B` | β | `E G` | γ |
| `E D` | δ | `E E` | ε | `E Z` | ζ |
| `E H` | η | `E V` | θ | `E I` | ι |
| `E K` | κ | `E L` | λ | `E M` | μ |
| `E N` | ν | `E X` | ξ | `E O` | ο |
| `E P` | π | `E R` | ρ | `E S` | σ |
| `E T` | τ | `E U` | υ | `E F` | ϕ |
| `E C` | χ | `E Y` | ψ | `E W` | ω |

## Device commands

| Sequence | Action |
|---|---|
| `U S B` | select USB output |
| `B L E` | select Bluetooth output |
| `R E S E T` | reset the keyboard |
| `B O O T` | enter the bootloader |

Reset and bootloader sequences change device state. Enter them deliberately.

The keymap contains 32 live leader sequences: 4 German, 24 Greek, and 4 device commands.
